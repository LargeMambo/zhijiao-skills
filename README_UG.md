## بۇ نېمە؟

**zhijiao-skills** بىر جۈملە بىلەنلا تولۇق دەرس ماتىرىياللىرى ياساپ بېرەلەيدىغان AI ماھارەت توپلىمى:

- 📊 **HTML ئىنتىراكتىپ دەرسلىك** — بىر ھۆججەت، تورسىزمۇ ئىشلەتكىلى بولىدۇ
- 📑 **PPTX دەرسلىك** — ئەنئەنىۋى پرويېكتور ۋە تاختايغا ماس
- 📝 **دەرس پىلانى** — Markdown فورماتى، باسما چىقارغىلى بولىدۇ
- 📋 **دەرىجىلىك مەشىق سوئاللىرى** — سىنىپتىكى مەشىق، ئۆي تاپشۇرمىسى ۋە بۆلۈم سىناق

**مىللىي رايون سىنىپلىرىغا ئالاھىدە لايىھەلەندى:**
- ✅ چوڭراق خەت، تېخىمۇ ئوچۇق مۇنداريجە
- ✅ ئاستا ئوقۇتۇش تېمپى — ھەر بەتتە بىرلا ئاساسىي بىلىم نۇقتىسى
- ✅ ئاساسىي بىلىم نۇقتىلىرى ئۈچۈن ئانىماتسىيە بارلىق تەسىرات
- ✅ ياسالغان دەرسلىكلەر تولۇق تورسىز ئىشلەيدۇ — تور يوق، پروگرامما ئورنىتىشقا كىرىشمەيدۇ
- ✅ ئۇيغۇرچە، تىبەتچە، قازاقچە قاتارلىق مىللىي تىل ئىزاھاتلىرىنى قوشۇشقا بولىدۇ
- ✅ ئۈچ دەرىجىلىك مەشىق، ئوقۇشى تۆۋەن ئوقۇغۇچىلار ئۈچۈن ئەنگىلدەتكەن نۇسخىسى بار

## ئورنىتىش يوللىرى

### بىرىنچى يول — چۈشۈرۈپ ئورنىتىش (پروگراممىلاشنى بىلمىگۈچىلەرگە ماس)

1. بۇ بەتتىكى يېشىل **Code** كۇنۇپكىسىنى بېسىڭ → **Download ZIP** تاللاڭ
2. چۈشۈرگەن ھۆججەتنى ئاچىڭ — `zhijiao-skills` ئاتلىق قىسقۇچنى كۆرىسىز
3. بۇ قىسقۇچنى Claude Code ماھارەت مۇندەرىجىسىگە كۆچۈرۈڭ:
   - **Windows:** `C:\Users\ئىسمىڭىز\.claude\skills\`
   - **Mac/Linux:** `~/.claude/skills/`
4. `claude` نى ئىشلەتكەندىن كېيىن `/skills` نى كىرگۈزۈپ ئورنىتىلغانلىقىنى دەلىللەڭ

### ئىككىنچى يول — npm ئارقىلىق ئورنىتىش (تەرەققىياتچىلارغا ماس)

```bash
# Claude Code ئورنىتىش
npm install -g @anthropic-ai/claude-code

# ماھارەت مۇندەرىجىسىگە كىرىش
cd ~/.claude/skills

# zhijiao-skills نى چۈشۈرۈش
git clone https://github.com/LargeMambo/zhijiao-skills.git
```

### ئۈچىنچى يول — Git ئارقىلىق ئورنىتىش

```bash
cd ~/.claude/skills
git clone https://github.com/LargeMambo/zhijiao-skills.git
```

### تۆتىنچى يول — باشقا AI قوراللىرىدا ئىشلىتىش

**SKILL.md** مەزمۇنىنى System Prompt (سىستېما تىلى) سۈپىتىدە ChatGPT، Claude.ai، Cursor، Ollama قاتارلىق قوراللاردا ئىشلىتەلەيسىز.

## كۆپ سورالغان سوئاللار

<details>
<summary><strong>س: پروگراممىلاشنى بىلمىسەم بولامدۇ؟</strong></summary>

بولىدۇ. ئورنىتىش قىسقۇچ كۆچۈرۈشتىن ئىبارەت. ئىشلىتىش ئوبدانلا ئانا تىلىڭىزدا سۆزلەشتىن ئىبارەت.

</details>

<details>
<summary><strong>س: ياسالغان دەرسلىككە تور كېرەكمۇ؟</strong></summary>

AI ياساش جەريانىدا تور كېرەك، ئەمما تامام بولغان `.html` ھۆججىتى تولۇق تورسىز ئىشلەيدۇ — USB غا كۆچۈرۈپ ھەر يەردە ئىشلىتەلەيسىز.

</details>

<details>
<summary><strong>س: ئۇيغۇرچە تېكىست توغرا كۆرسىتىلىدۇ؟</strong></summary>

ھەئە. ئۇيغۇرچە RTL (ئوڭدىن سولغا) يازما بولغاچقا، بۇ ماھارەت `dir="rtl"` ئاپتوماتىك قوللايدۇ. ئاساسىي ئاتالغۇلار سىستېمىدا بار.

</details>

<details>
<summary><strong>س: مىللىي تىل تەرجىمىلىرى توغرامۇ؟</strong></summary>

ئومۇمەن توغرا، ئەمما شىۋە پەرقى بار. مۇھىم مەزمۇنلارنى يەرلىك تىل بىلىدىغان كىشىگە تەكشۈرتۈپ بېقىڭ.

</details>

<details>
<summary><strong>س: ChatGPT ياكى Codex دا قانداق ئىشلىتىمەن؟</strong></summary>

`SKILL.md` مەزمۇنىنى كۆچۈرۈپ System Prompt غا چاپلاڭ. ياخشى نەتىجە ئالىش ئۈچۈن `references/` مۇندەرىجىسىدىكى ھۆججەتلەرنىمۇ بىرگە قوشۇپ بېرىڭ.

</details>
