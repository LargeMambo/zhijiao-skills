## འདི་ཅི་ཡིན་ནམ？

**zhijiao-skills** ནི་AI ཐབས་ལམ་གྱི་ལག་ཆས་ཚོགས་པ་ཞིག་ཡིན་པས། སློབ་ཁྲིད་མཁན་གྱི་དགེ་རྒན་ཚོས་ཚིག་གཅིག་གིས་སློབ་ཁྲིད་ཀྱི་ཡོ་ཆས་ཧ་ཅང་ལེགས་པར་བཟོ་ཐུབ།

- 📊 **HTML སྤྱོད་ཚུལ་ལྡན་པའི་སློབ་ཁྲིད་ཡོ་ཆས** — ཡིག་ཆ་གཅིག་པ་ཡིན། ལྟ་བཤེར་མི་དགོས་པར་ཁ་ཕྱི་རུ་བཀོལ་ཆོག
- 📑 **PPTX སློབ་ཁྲིད་ཡོ་ཆས** — རྒྱུན་ལྡན་གྱི་གཞི་ས་གཅེར་འཕར་དང་གྱང་ཐོག་གཅིག་སྦྲེལ་མཁར་ལ་མཐུན་པ
- 📝 **སློབ་ཁྲིད་ལག་ཁྱེར** — Markdown རྣམ་གཞག། འབྲི་སྡུར་ལ་གྲ་ལྡན།
- 📋 **རིམ་པ་ཐོ་ཆུང་ལྡན་པའི་དྲི་བ་ཁུར** — ཐུགས་འཁྲུག་གི་སྦྱོང་བརྡར། གྲས་སྦྱོར་འགན་ཁུར། དང་ས་ཆ་ཚར་གཅིག་གི་ཚོད་ལྟ།

**མི་རིགས་མང་བོའི་ཁྲིད་ཡུལ་ལ་ཁྱད་པར་དུ་བཟོས་པ:**
- ✅ ཡི་གེ་ཆེས་ཆེར། བཀོད་གཞག་ཆེས་གསལ།
- ✅ སློབ་ཁྲིད་ཀྱི་འགྲོས་ཉིར་བ་— ཤོག་ལེབ་རེར་དོན་ཆེན་ཤེས་བྱ་གཅིག
- ✅ གལ་ཆེའི་ཤེས་བྱའི་ཚེས་སུ་གཞི་སྐར་གྱི་བྱིབ་བྱིབ་རྒྱབ་ཀྱི་བྱིབ་བྱིབ་རྒྱགས་པ
- ✅ བཟོས་ཟིན་པའི་སློབ་ཁྲིད་ཡོ་ཆས་ཁ་ཕྱི་རུ་བཀོལ་ཆོག — དྲ་བ་མི་དགོས། ལག་ཆས་གཞན་གསར་འཛུགས་མི་དགོས།
- ✅ བོད་སྐད། ཡུ་གུར་སྐད། ཀ་ཛག་སྐད་སོགས་ཀྱི་ཡིག་གཟུགས་ཀྱི་མཆན་བུ་ཡིས་སྣོན་ཐུབ།
- ✅ གོ་རིམ་གསུམ་གྱི་དཀའ་ཚེགས་ལྡན་པའི་སྦྱོང་བརྡར། སྦྱངས་དཀའ་བའི་སློབ་མར་ཕན་པའི་སྟབས་བདེའི་པར་གཞིར་ཡིས།

## གཞི་བཀལ་ཐབས།

### ཀ) ཕབ་ལེན་བྱས་ཏེ་བཞག་པ (ལག་རྩལ་མི་དགོས་པར་ཕན་ཐོག་ཆེས་ཆེ།)

1. བང་ཆེན་པོའི་**Code** མདའ་དེ་གཅལ་ → **Download ZIP** གདམ།
2. ཕབ་ལེན་བྱས་པའི་ཡིག་ཆ་གཤོར། `zhijiao-skills` ཟེར་བའི་ཡིག་ཆའི་སྒྲོམ་ཞིག་རྙེད་ངེས།
3. སྒྲོམ་འདི་Claude Code ལག་ཆས་ཀྱི་ས་གནས་སུ་བཤུ།
   - **Windows:** `C:\Users\ཁྱེད་ཀྱི་མིང་\.claude\skills\`
   - **Mac/Linux:** `~/.claude/skills/`
4. བཀའ་ལམ་ `claude` ཞུགས་ཏེ་ `/skills` ཞུས་ན་གཞི་བཀལ་ཐུབ་མིན་ཤེས་ཐུབ།

### ཁ) npm བཀོལ་ཏེ་གཞི་བཀལ་བ (ལག་རྩལ་ཅན་གྱི་མི་ལ་ཕན་ཐོག་ཆེ།)

```bash
# Claude Code གཞི་བཀལ་བ
npm install -g @anthropic-ai/claude-code

# ལག་ཆས་ཀྱི་ས་གནས་སུ་འགྲོ།
cd ~/.claude/skills

# zhijiao-skills ཕབ་ལེན།
git clone https://github.com/LargeMambo/zhijiao-skills.git
```

### ག) Git བཀོལ་ཏེ་གཞི་བཀལ་བ

```bash
cd ~/.claude/skills
git clone https://github.com/LargeMambo/zhijiao-skills.git
```

### ང) ཁ་ཕྱི་གཞན་གྱི་AI ལག་ཆས་ལ་བཀོལ་སྤྱོད་

**SKILL.md** ཀྱི་ནང་དོན་ཧ་ཅང་System Prompt (ལམ་སྟོན་ཚིག་) ཡིན་ཏེ་ ChatGPT། Claude.ai། Cursor། Ollama སོགས་ལ་བཀོལ་སྤྱོད་ཐུབ།
