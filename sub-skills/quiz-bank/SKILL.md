# 随堂题库生成子技能

## 工作流

### Step 1 — 确认参数

收集信息：
- 学科、年级、课题/知识点范围
- 题库用途（随堂练习 / 课后作业 / 单元检测 / 期末复习）
- 题量需求（参见下方默认配置）
- 难度分布（默认：基础60% / 提升30% / 拓展10%）
- 是否需要解析（默认：是）

### Step 2 — 默认题量配置

根据用途自动选择默认题量，教师可自行调整：

| 用途 | 总题量 | 参考时长 |
|---|---|---|
| 随堂小练（课中穿插）| 5-8题 | 5-8分钟 |
| 课后作业 | 12-18题 | 20-30分钟 |
| 单元小测 | 25-35题 | 40分钟 |
| 期末复习卷 | 40-55题 | 60-90分钟 |

### Step 3 — 题型配置

根据学科和年级自动搭配题型，也可由教师指定：

**通用题型（所有学科）：**
- 选择题（单选/多选）
- 判断题（正确 / 错误）
- 填空题
- 简答题

**理科专用：**
- 计算题（步骤分）
- 公式填写题
- 图形/表格分析题

**语文专用：**
- 字词注音/组词
- 词语填空
- 句子改错/扩写/缩写
- 阅读理解（短文 + 问答）
- 古诗默写

**英语专用：**
- 单词拼写/翻译
- 句型转换
- 完形填空（小段）
- 阅读理解

**低年级友好题型（1-3年级）：**
- 连线题（词语与图片/概念配对）
- 排序题（步骤/事件顺序）
- 涂色/圈画题（以文字描述代替）
- 看图说话/写话

### Step 4 — 生成题库

**格式规范：**

```markdown

**学科：** [学科]  **年级：** [年级]  **课题：** [课题]
**建议用时：** X 分钟  **总分：** XX 分

---

## 第一部分：选择题（每题X分，共X题）

**1.** [题目内容]
   A. [选项]　B. [选项]　C. [选项]　D. [选项]

   A. [选项]　B. [选项]　C. [选项]　D. [选项]
```

### Step 5 — 文化适配提示

在涉及应用题场景时，优先选用以下背景：
- 数学：买馕/哈密瓜/牛羊计数/葡萄园面积/坎儿井流量/藏区牧场面积
- 语文：描述天山/塔里木/草原/雪域高原的文段
- 英语：民族节日/特色食物/传统服饰等话题
- 科学：沙漠植物/高原气候/太阳能利用

### Step 6 — 输出

Markdown 格式输出，文件名：`[课题名称]-题库.md`

输出完毕后，询问：
> "题库已生成，包含基础层、提升层、拓展层三套难度，并附有学困生简化版。
> 需要将练习题整合到课件 PPT 中吗？"
