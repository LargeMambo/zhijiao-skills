# 输出质检清单

在输出任何成品文件前，逐项核查以下清单。

---

## HTML 课件（ppt-html）

### 视觉与可读性
- [ ] 所有正文字号 ≥ `1.8rem`
- [ ] 所有标题字号 ≥ `3.2rem`
- [ ] 题目和关键词字号 ≥ `2.4rem`
- [ ] 关键词有红底白字高亮标注
- [ ] 配色符合 WCAG AA 对比度（文字与背景对比度 ≥ 4.5:1）
- [ ] 每页有清晰的视觉焦点，不存在信息堆砌

### 离线完整性（最重要）
- [ ] HTML 文件中 **没有** `<script src="http...">` 外链
- [ ] HTML 文件中 **没有** `<link href="http...">` CDN 字体链接
- [ ] HTML 文件中 **没有** `<img src="http...">` 外链图片
- [ ] 所有 SVG 已 inline（`<svg>` 标签直接在 HTML 中）
- [ ] Motion One JS 已 inline（`<script>` 标签内）
- [ ] 用 `file://` 协议在浏览器打开，功能完整，无 404 错误

### 动效
- [ ] `data-fx="particles"` 仅在 L01 封面使用
- [ ] `data-fx="confetti"` 仅在 L12 收尾使用
- [ ] 其他页面使用 `data-anim="fade-in"` 或规定的元素级动效
- [ ] 同一页面 `data-anim` 类型 ≤ 3 种

### 导航
- [ ] 键盘左右箭头可以翻页
- [ ] 底部导航点可点击
- [ ] ESC 键可以打开幻灯片概览

### 双语注释（如启用）
- [ ] 维吾尔语/哈萨克语文本设置了 `dir="rtl"`
- [ ] 注释仅在概念讲解页出现，题目页无注释
- [ ] 注释字号不大于正文字号

---

## 教案（lesson-plan）

- [ ] 包含所有八个模块（不得省略）
- [ ] 学情分析中有明确的学困生照顾策略
- [ ] 教学过程表格中每个环节有时间、教师活动、学生活动、设计意图
- [ ] 例题和练习题有分层（⭐ / ⭐⭐ / ⭐⭐⭐）
- [ ] 作业有基础层/提升层/拓展层三档
- [ ] 文化适应举例贴近西部生活场景
- [ ] 语言简洁，句子不超过 20 字

---

## 题库（quiz-bank）

- [ ] 题量符合所选用途的默认配置
- [ ] 每道题标注了难度（⭐ / ⭐⭐ / ⭐⭐⭐）
- [ ] 每道题标注了考查知识点
- [ ] 所有题目都有参考答案
- [ ] ⭐⭐⭐ 拓展题有文化适应场景（新疆/西藏/西部元素）
- [ ] 附有学困生简化版（仅含 ⭐ 基础题）
- [ ] 题目语言简洁，无歧义

---

## 通用检查

- [ ] 文件命名规范：`[课题名称]-课件.html` / `[课题名称]-教案.md` / `[课题名称]-题库.md`
- [ ] 输出完毕后询问用户是否需要其他配套材料
