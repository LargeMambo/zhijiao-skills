## 这是什么？

**支教技能组**是一个 AI 提示词技能包，安装后可以帮助支教教师用一句话完成备课工作：

- 📊 生成 **HTML 互动课件**（一个文件，可在任何设备离线使用）
- 📑 生成 **PPTX 传统课件**（适配白板一体机、老式投影仪）
- 📝 生成 **教案**（教学设计，Markdown 格式，可直接打印）
- 📋 生成 **分层题库**（随堂练习、课后作业、单元检测）

**专为少数民族地区设计：**
- ✅ 字体更大，排版更清晰
- ✅ 教学节奏慢，每页聚焦一个核心知识点
- ✅ 关键知识点动效突出（3D 翻转、弹跳高亮、彩纸庆祝）
- ✅ 生成的课件完全离线——不需要网络，不需要安装软件
- ✅ 可选添加维吾尔语、藏语、哈萨克语等少数民族语言小字注释
- ✅ 题库分三层难度，专门设计了学困生简化版

## 安装指南

> 📌 **如果您缺少编程基础：** 下面的步骤可能看起来有点陌生，但非常简单，跟着一步步做就好。

### 前提条件

你需要先安装好 **Claude Code** 或其他支持技能组的 AI 工具。

**检查方法：** 打开终端（Windows 按 `Win+R` 输入 `cmd`，Mac 按 `Cmd+空格` 输入 `terminal`），输入：
```
claude --version
```
如果看到版本号（如 `1.x.x`），说明已安装好。

如果还没有安装，请参考 [Claude Code 官方安装文档](https://docs.anthropic.com/zh-CN/docs/claude-code/getting-started)。

---

### 方法一：直接下载安装（推荐给非程序员）

**第一步：下载技能包**

访问本项目页面，点击绿色的 **Code** 按钮，选择 **Download ZIP**，下载后解压到你喜欢的位置。

解压后你会看到一个叫 `zhijiao-skills` 的文件夹。

**第二步：安装到 Claude Code**

把这个文件夹复制到 Claude Code 的技能目录：

- **Windows 系统：**
  ```
  C:\Users\你的用户名\.claude\skills\
  ```
  把 `zhijiao-skills` 文件夹复制进去，最终路径是：
  ```
  C:\Users\你的用户名\.claude\skills\zhijiao-skills\
  ```

- **Mac 系统：**
  ```
  /Users/你的用户名/.claude/skills/
  ```
  （如果看不到 `.claude` 文件夹，请在 Finder 中按 `Cmd+Shift+.` 显示隐藏文件）

**第三步：验证安装**

打开终端，输入：
```
claude
```
进入 Claude Code 后，输入：
```
/skills
```
如果看到列表里有 `zhijiao-skills`，说明安装成功！

---

### 方法二：使用 Git 安装（给熟悉命令行的用户）

打开终端，执行：

```bash
# 进入技能目录（Windows）
cd %USERPROFILE%\.claude\skills

# 或 Mac/Linux
cd ~/.claude/skills

# 克隆本仓库
git clone https://github.com/你的用户名/zhijiao-skills.git
```

---

### 方法三：通用 AI 工具使用

本技能组不依赖 Claude Code 独有功能，可以在以下工具中直接使用：

| 工具 | 使用方式 |
|---|---|
| **Claude.ai 网页版** | 上传 `SKILL.md` 文件到对话，或将其内容粘贴为系统提示 |
| **ChatGPT / GPT-4** | 将 `SKILL.md` 内容粘贴到 System Prompt |
| **Cursor / Windsurf** | 将 `SKILL.md` 添加为项目规则文件 |
| **本地 AI（Ollama 等）** | 将 `SKILL.md` 内容作为系统提示使用 |

> 使用通用 AI 时，建议同时附上 `references/` 目录下的参考文件，以获得更好的输出效果。

## 使用方法

安装完成后，直接用自然语言和 AI 对话即可，**不需要记忆任何命令**。

### 快速上手示例

以下是一些直接可以复制使用的提示词：

---

**🎯 只做课件（HTML 版）**
```
帮我做一个《春》的语文课件，六年级，人教版
```

---

**🎯 只做教案**
```
我要备课《牛顿第一定律》，初中物理八年级，
班里维吾尔族学生较多，请帮我写一份详细教案
```

---

**🎯 只出题**
```
出一套《100以内加减法》的随堂练习题，
二年级数学，要有基础题、提升题和拓展题，
题量是课后作业的量
```

---

**🎯 全套备课（教案+课件）**
```
我下周要讲《水的三态变化》，四年级科学，
请帮我做一整套备课材料：教案和HTML课件
```

---

**🎯 带少数民族语注释的课件**
```
帮我做《认识钟表》的数学课件，二年级，
请在关键词旁边加上维吾尔语的小字注释
```

---

**🎯 指定风格**
```
帮我做《植物的光合作用》课件，初一生物，
配色用草绿主题，课件要用PPTX格式（我的教室只有老式投影仪）
```

---

### 工作流程

技能组会按以下流程工作，你只需要回答它的问题：

```
1. AI 确认：学科、年级、课题、教材版本
   （没提供的会逐一询问你）

2. AI 规划：列出课件页面结构清单，供你确认
   （可以说"调整第5页"或"没问题，继续"）

3. AI 生成：教案/课件/题库

4. AI 询问：是否需要转换为 PPTX 格式？
   是否需要添加双语注释？
   是否需要配套题库？
```

---

### 输出文件说明

所有文件会保存在你当前工作目录下：

| 文件 | 格式 | 用途 |
|---|---|---|
| `课题名-课件.html` | 单文件 HTML | 直接用浏览器打开，支持离线使用 |
| `课题名-课件-pptx版.pptx` | PowerPoint | 在老式投影仪/白板一体机上使用 |
| `课题名-教案.md` | Markdown | 用任何文本编辑器打开，可打印 |
| `课题名-题库.md` | Markdown | 可打印为学生练习卷 |

**打开课件的方式：**
直接双击 `.html` 文件，用任何浏览器（Chrome、Edge、Safari）打开即可。

**打印教案的方式：**
用 VS Code、Typora、或任何 Markdown 编辑器打开 `.md` 文件，然后打印或导出为 PDF。

## 功能详解

### HTML 课件特性

| 功能 | 说明 |
|---|---|
| 完全离线 | 单个 HTML 文件，无需网络，U盘拷贝即可用 |
| 键盘翻页 | ← → 方向键翻页，ESC 键打开所有页面概览 |
| 触摸支持 | 手机、平板触摸滑动翻页 |
| 关键词高亮 | 核心概念红底白字标注 |
| 重点动效 | 封面粒子、例题逐步揭示、结尾彩纸 |
| 答案折叠 | 跟练题目点击"查看答案"才显示 |

### 配色主题

| 主题 | 适用场景 |
|---|---|
| 🟠 暖橙（默认） | 数学、小学课堂 |
| 🔵 天蓝 | 语文、英语、初中课堂 |
| 🟢 草绿 | 科学、生物、地理 |
| ⬛ 高对比黑白 | 老旧投影仪、强光教室 |

在提示词中说明即可切换，如：`配色用天蓝主题`

### 少数民族语言注释

在提示词中说明需要哪种语言：

```
帮我做课件，在关键词旁边加上 [语言] 的注释
```

支持语种：

| 语种 | 代码 | 地区 | 文字方向 |
|---|---|---|---|
| 维吾尔语 | ug | 新疆 | 从右向左（RTL） |
| 藏语 | bo | 西藏、青海 | 从左向右 |
| 哈萨克语 | kk | 新疆 | 从右向左 |
| 蒙古语 | mn | 内蒙古、新疆 | 从左向右 |
| 柯尔克孜语 | ky | 新疆 | 从右向左 |

> **注意：** 少数民族语言词汇由 AI 生成，重要课程请请当地教师审核。

## 常见问题

**Q：我不懂编程，能用吗？**
A：完全可以。安装步骤只是复制文件夹，使用时只需要用中文描述你的需求，像平时和人说话一样。

**Q：生成的课件需要网络吗？**
A：不需要。生成过程需要联网（AI 需要网络），但生成完毕的 `.html` 文件是完全离线的，可以拷贝到 U 盘在任何地方使用。

**Q：支持哪些浏览器？**
A：Chrome、Edge、Firefox、Safari 均可。在学校老旧电脑上建议使用最新版 Chrome 或 Edge（免费下载）。

**Q：生成 PPTX 需要安装 Office 吗？**
A：生成 PPTX 文件需要 AI 工具环境中有相关支持（Claude Code 的 pptx skill）。如果不行，建议直接使用 HTML 版课件。

**Q：教案里的少数民族语翻译准确吗？**
A：AI 生成的翻译准确率较高，但存在方言差异。涉及重要概念，请让当地会说少数民族语的同事帮忙核对。

**Q：可以修改生成的课件吗？**
A：可以。HTML 课件是标准的网页文件，可以用任何文本编辑器（如 VS Code、记事本）打开修改。

**Q：如何更新技能组？**
A：删除旧的 `zhijiao-skills` 文件夹，重新下载安装即可。
